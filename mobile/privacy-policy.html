Política de Privacidade

Última atualização: Março de 2026

1. Informações Coletadas

O aplicativo Dieta.AI coleta os seguintes dados fornecidos voluntariamente pelo usuário: nome, peso, altura, idade, sexo, nível de atividade física e objetivo nutricional. Esses dados são utilizados exclusivamente para gerar dietas personalizadas via inteligência artificial.

2. Uso dos Dados

Os dados são processados pela API do Google Gemini para gerar recomendações nutricionais. O histórico de dietas é armazenado no Firebase Firestore, vinculado a um identificador anônimo (sem e-mail ou dados pessoais identificáveis).

3. Publicidade

Utilizamos o Google AdMob para exibir anúncios. O AdMob pode coletar identificadores de publicidade (GAID) conforme as políticas do Google. O usuário pode optar por anúncios não personalizados ao iniciar o aplicativo.

4. Compartilhamento

Não compartilhamos, vendemos ou transferimos dados pessoais a terceiros, exceto conforme necessário para operar os serviços descritos (Google Cloud, Firebase, AdMob).

5. Segurança

Utilizamos Firebase Authentication, Firestore Security Rules e App Check para proteger os dados contra acessos não autorizados.

6. Direitos do Usuário (LGPD)

Conforme a Lei Geral de Proteção de Dados (Lei 13.709/2018), o usuário tem direito a: acesso, correção, exclusão e portabilidade dos seus dados. Para exercer esses direitos, entre em contato pelo e-mail abaixo.

7. Contato
